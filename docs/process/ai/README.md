# Processo de Desenvolvimento com Assistentes de IA

Esta pasta contém documentos "meta", ou seja, que descrevem **como trabalhamos** com assistentes de IA neste projeto.

**Estes documentos não descrevem a arquitetura do sistema**, mas sim as diretrizes, prompts e perguntas que usamos para guiar as IAs durante o desenvolvimento, revisão e automação de tarefas.

- **`AI_TOOLING.md`**: Define o papel de cada ferramenta de IA no nosso fluxo de trabalho.
- **`CODEX_GUIDE.md`**: O "super-prompt" ou guia mestre para a IA que atua como assistente de implementação.
- **`CODEX_QUESTIONS.md`**: Um registro histórico de perguntas feitas para a IA para guiar o design inicial.
