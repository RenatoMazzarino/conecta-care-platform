# Repositório Antigo (Legado)

Esta pasta contém artefatos históricos do projeto anterior ou versões legadas do sistema.

## Aviso Importante

**O conteúdo desta pasta serve APENAS como referência histórica.**

-   **NÃO** utilize o arquivo `schema_current.sql` como base para novas implementações ou decisões de arquitetura. O schema atual e oficial do projeto reside nas migrações em `supabase/migrations`.
-   **NÃO** copie código ou lógica daqui sem validar contra os contratos atuais em `docs/contracts`.

O objetivo de manter estes arquivos é permitir a consulta de regras de negócio antigas para fins de migração e entendimento de contexto, mas eles **não representam a verdade atual do sistema**.
